What is ClassCash?


ClassCash is a class rewards system designed to provide students with rewards for good behavior and teachers the ultimate customization they want with a rewards system.

Built for stability


While other rewards systems like Percle struggle with stability and performance as they are built on the unoptimized Glide, ClassCash is designed to deliver unprecedented performance.
Built on HTML5 and JavaScript on the frontend, and PHP 8 on the backend, ClassCash will never crash or slow down, no matter how many features are implemented.

Powered by the real stuff


Unlike Percle, ClassCash is built on a MySQL database with military-grade encryption, unlike Percle that is built on a simple Google Sheets database.
This allows for more scalability if more features are to come, and ensures maximum data management performance.

No glitches


We know you hate glitches and we do as well. At ClassCash, we make sure to test every single feature, even edge cases, before we deploy it.
Unlike Percle, this ensures that there are no game-changing glitches like negative funds.

Ultimate customizability


ClassCash is designed to be as user friendly as possible. This means that we have a whole array of settings you can toggle to foster the class rewards environment you've always wanted.
Even though most rewards systems make a lot of decisions for the teacher which can be frustrating, ClassCash allows the teacher to make every decision.


ClassCash, your rewards superpower.
